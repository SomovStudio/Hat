--------------------------------------------------------------------
Таблица: groups - группы (разбивка автотестов на группы)
--------------------------------------------------------------------
- id (идентификатор группы)
- name (имя)
- action (включена/отключена)


--------------------------------------------------------------------
Таблица: tests - тесты (тут перечень всех автотестов и их настройки)
--------------------------------------------------------------------
- id (идентификатор теста)
- group (идентификатор группы к которой привязан тест)


--------------------------------------------------------------------
Таблица: logs - логи (сюда сохраняются строки шагов из консоли)
--------------------------------------------------------------------
- id (идентификатор лога)
- test (идентификатор теста к которой привязан лог)
- date (дата лога)
