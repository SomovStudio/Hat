ПУБЛИЧНЫЕ МЕТОДЫ И КОНСТАНТЫ


consoleMsg(string message) - отправить сообщение в консоль
clearMessage() - очистить сообщения в таблице теста
sendMessage(string step, string status, string comment, int image) - отправить сообщение в таблицу теста
editMessage(int index, string step, string status, string comment, int image) - изменить сообщение в таблице теста

testBegin() - подготовка теста
testEnd() - завершение теста

goToUrl(string url) - загрузить страницу
getURL() - получить текущий url страницы
getHtmlElement(string locator) - получить элемент
clickHtmlElement(string locator) - нажать на элемент
setValueFromInput(string locator, string value) - вставить значение в input
getValueFromInput(string locator) - получить значение из input
getValueFromTextarea(string locator) - получить значение из textarea
getTextFromHtmlElement(string locator) - получить текст из элемента
wait(int sec) - ожидание

assertEquals(string expected, string actual)
assertNotEquals(string expected, string actual)
assertTrue(bool condition)
assertFalse(bool condition)
