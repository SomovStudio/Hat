Обновление браузера Hat 1.1.0

Список внесённых изменений:

- Добавлена документация
- Обновлен плагин WebView до версии 2.1.0.1245.22
- В desktop режиме консоль не отображается
- Исправлен баг при повторной загрузке страницы
- Отчеты о прохождении автотестов в html формате со скриншотами
- Режим отладки при выполнении метода ExecuteJavaScriptAsync
- Создан класс HTMLElement и внесены изменения в метод GetElementAsync
- Добавлена возможность запуска группы автотестов
- Добавлена поддержка XPath в методах
- Добавлены методы для работе с REST запросами
- Добавлен метод BrowserGetErrorsAsync для получения списка ошибок и предупреждений браузера
- Добавлен метод BrowserGetNetworkAsync для получения в формате json всех текущих событий из network
- Изменены методы BrowserGetUserAgentAsync и BrowserSetUserAgentAsync
- Добавлены методы GetAttributeFromElements и SetAttributeInElements
- Добавлены методы GetHtmlFromElement и SetHtmlInElement
